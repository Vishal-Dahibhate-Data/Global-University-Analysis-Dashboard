 🎓 Global University Success Analysis

## Comprehensive Analysis of Global University Rankings Using Power BI, SQL & Excel

A Business Intelligence project that analyzes university performance across the globe by integrating data from multiple international ranking systems. The project leverages SQL, Excel, and Power BI to uncover trends, ranking factors, student demographics, and institutional performance metrics that influence university success.

## 📌 Project Overview

The Global University Success Analysis project provides a comprehensive evaluation of universities worldwide using data from multiple ranking systems, including:

* Center for World University Rankings (CWUR)
* Times Higher Education World University Rankings (THE)
* Shanghai Ranking (ARWU)

The project aims to identify key factors contributing to university rankings and evaluate institutional performance through interactive dashboards and data-driven insights.

## 🎯 Business Problem

Universities are evaluated using different ranking methodologies, making it challenging to understand the factors driving institutional success.

This project addresses the following questions:

* Which countries host the highest-performing universities?
* What factors most influence university rankings?
* How do student demographics impact university performance?
* What trends can be observed across different years and ranking systems?
* How do international students and student-staff ratios affect rankings?

## 🎯 Project Objectives

* Compare university rankings across multiple ranking systems.
* Identify critical factors influencing university success.
* Analyze student demographics and enrollment trends.
* Evaluate university performance at country and institutional levels.
* Examine ranking changes over time.
* Perform correlation analysis between ranking metrics and university characteristics.
* Build interactive dashboards for strategic decision-making.

* Significant growth in international student enrollment across universities.
* Increase in female student participation over the years.
* Strong-performing universities consistently rank highly across ranking systems.

* International student representation positively impacts global visibility.

* Institutional resources contribute significantly to ranking performance.
* Student demographics influence university competitiveness.
